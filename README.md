# Mcworlde
# 🧱 MCWorld Web Editor

A web-based Minecraft Bedrock world editor with real `.mcworld` support.

## 🚀 Features

- Import `.mcworld`
- Extract LevelDB
- Read chunk data
- 3D editor frontend
- Export back to `.mcworld`

## 🛠 Setup

### Backend

```bash
cd backend
npm install
node server.js
